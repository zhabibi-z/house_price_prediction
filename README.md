# House Price Prediction

Machine learning model for predicting residential property prices. Using Linear Regression.

## Overview

This project builds a regression pipeline to estimate house prices based on structural and location features.

## Project Structure

```
├── data/          # Raw and processed datasets
├── notebooks/     # Exploratory data analysis
├── src/           # Model training and evaluation code
├── outputs/       # Trained models and reports
└── requirements.txt
```

## Getting Started

```bash
pip install -r requirements.txt
```
